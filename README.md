### 🎗️ Breast Cancer Detection — KNN Classifier

A machine learning web app that classifies breast tumors as **Malignant** or **Benign** using the K-Nearest Neighbors algorithm.

#### 🚀 Live Demo
[Click here to view the app]

#### 📊 Features
- Tumor classification using KNN 
- Interactive sliders for 10 key tumor measurements
- Deep Dive page with radar chart analysis
- Breakdown of every measurement

#### 🗂️ Dataset
Wisconsin Breast Cancer Dataset — 569 samples, 30 features, 2 classes.

#### 🛠️ Tech Stack
- Python, Scikit-learn, Streamlit
- Pandas, NumPy, Matplotlib
